#### **Relevant notes**:
**There are 3 Tree ring phenotypes that we can use for association genetics:**
- Type A: basal area increment (BAI)
- Type B: Growth response to past extreme climatic events
- Type C: climate sensitivity which is the relationship between growth time series and climatic fluctuations
